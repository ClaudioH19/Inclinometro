1. Metricas, formulas y metodo

Metodo breve: el ESP32 registra muestras crudas con sample_time_us relativo al encendido. El servidor calcula magnitudes, orientacion y resumen estadistico por sesion filtrada.

Sin cargar.

Tasa de muestreoHz = 1e6 / mean(delta_sample_time_us)
Periodo de muestrams = mean(delta_sample_time_us) / 1000
Aceleracion total|a| = sqrt(ax^2 + ay^2 + az^2)
Velocidad angular|w| = sqrt(gx^2 + gy^2 + gz^2)
Pitchatan2(-ax, sqrt(ay^2 + az^2))
Rollatan2(ay, az)
RPM estimadarpm = |gz_dps| / 6
Gravedad residualg_res = sqrt(mean(ax)^2 + mean(ay)^2 + mean(az)^2)
