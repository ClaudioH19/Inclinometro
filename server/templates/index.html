fórmulas y método

El ESP32 registra muestras crudas con sample_time_us relativo al encendido. El servidor calcula magnitudes, orientación y resumen estadístico por sesión filtrada.

Tasa de muestreoHz = 1e6 / mean(Δt_us)
Periodo de muestrams = mean(Δt_us) / 1000
Aceleración total|a| = √(ax² + ay² + az²)
Velocidad angular|ω| = √(gx² + gy² + gz²)
Pitchatan2(−ax, √(ay²+az²))
Rollatan2(ay, az)
RPM estimadarpm = |gz_dps| / 6
Gravedad residual√(x̄ₐ² + ȳₐ² + z̄ₐ²)
