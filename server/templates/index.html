Formulas

Resumen corto de como se calcula cada valor del panel.

RPM rpm = |gyro_axis_dps| / 6
Velocidad angular |w| = sqrt(gx^2 + gy^2 + gz^2)
Aceleracion |a| = sqrt(ax^2 + ay^2 + az^2)
Pitch atan2(-ax, sqrt(ay^2 + az^2))
Roll atan2(ay, az)
Gravedad residual g_res = sqrt(mean(ax)^2 + mean(ay)^2 + mean(az)^2)
Residual g*h g_res * horas_operacion
Horas de operacion (ultimo_t - primer_t) / 3.6e9
